### RMI in Java
You must be able to produce code in Java using RMI which creates a simple distributed system.
Generally you need:
* Interface on Server
* Implementing Class
* Main Class that puts object into registry
* Client Main Class
... and for callbacks ...
* Interface on Client
* Implementing Class
* Data Structure on server to store clients to callback to
